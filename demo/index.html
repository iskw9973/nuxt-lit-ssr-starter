外部サイト埋め込みデモ

Nuxt なしの純粋な HTML ページです。demo/dist/wc.js を読み込むだけで Web Component が動作します。

lit-card

外部サイトからの埋め込み

このカードは Nuxt を使わず、スタンドアロンの JS バンドルで描画されています。 Nuxt 側では DSD として SSR されますが、こちらはクライアント JS のみで動作します。

※ Nuxt アプリでは <template shadowrootmode="open"> としてサーバーレンダリングされ、JS 読み込み前から表示されます。

lit-card（elevated なし）

フラットカード

elevated 属性を外したバリアントです。

lit-counter

初期値は HTML 属性で渡します。JS 読み込み後にインタラクティブになります。

埋め込みコード

<!-- 1. バンドルを読み込む -->
<script type="module" src="https://your-cdn.example/wc.js"></script>

<!-- 2. タグをそのまま使う -->
<lit-card elevated>
  <span slot="heading">タイトル</span>
  <p>本文テキスト</p>
</lit-card>

<lit-counter count="0" label="クリック"></lit-counter>
